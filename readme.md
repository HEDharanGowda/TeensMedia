Teens Media Application 🛡️📱

A safe and controlled social media platform for teenagers, focused on content moderation, privacy, and a healthy online environment.

🚀 Overview

Teens Media is a media-sharing application designed exclusively for teens, where safety comes first.
The platform prevents harmful or inappropriate content, enforces strict moderation rules, and provides admins with tools to monitor activity.

This project is built as a learning + real-world MVP, following modern full-stack development practices.

🎯 Key Features
* Secure user authentication (JWT-based)
* Image & media upload with content moderation
* Automatic blocking of unsafe content
* User violation tracking & account disabling
* Admin panel for reviewing flagged posts
* Safe feed showing only approved content
* REST API–based architecture
* Scalable & modular codebase
